# Unit conversion

Compute directly — no tools, so any model can do it. Common factors:

- **Length:** 1 mi = 1.609344 km · 1 ft = 0.3048 m · 1 in = 2.54 cm.
- **Weight:** 1 lb = 0.453592 kg · 1 oz = 28.3495 g.
- **Temperature:** C = (F−32)×5/9 · F = C×9/5+32 · K = C+273.15.
- **Volume:** 1 US gal = 3.78541 L · 1 US cup = 236.588 mL.
- **Speed:** 1 mph = 1.609344 km/h.

## Rules
- Show the result with the correct unit, rounded sensibly; include the formula if it helps.
- **Currency** rates change — `web_search` the current rate (or state you used an approximate one).
