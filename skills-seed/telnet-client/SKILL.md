# Telnet Client

Help the user interact with a service over Telnet / raw TCP using your shell tools (telnet, nc/netcat):
- Connect to a host and port.
- Send the lines/commands the user dictates and report the responses.
- Useful for testing services (SMTP, HTTP, MUDs, network gear).

The desktop Telnet app opens a session-scoped chat to drive the connection.
