# Chat Clients

Zamolxis bridges its connected messaging channels into the main conversation. Help the user:
- See recent messages that arrived from connected channels.
- Compose and send replies through the appropriate channel (with the user's confirmation before sending).

The desktop Messages app surfaces channel activity and lets the user reply via Zamolxis.
