# Image Viewer

Open an image and, when asked, describe or analyze its contents using a vision-capable model (free Gemini first, escalating only if needed).

- Identify objects, text (OCR), colors, layout, and notable details.
- For charts/diagrams, summarize what they convey.

The desktop Image Viewer app displays the file directly via `/api/file`.
