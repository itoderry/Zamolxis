# Ebook Reader

Open an e-book (EPUB or PDF) and help the user read it: summarize chapters, find passages, explain terms, or extract quotes.

- For text questions, read the book's text (extract from EPUB/PDF) and answer from it.
- Keep quotes short and attributed.

The desktop Ebook Reader app renders EPUB (paginated) and PDF directly via `/api/file`.
